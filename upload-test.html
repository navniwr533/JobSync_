Resume Upload Test

Instructions

  1. Open browser console (F12) to see debug logs
  2. Make sure you're logged in to JobSync
  3. Upload a resume file (any text file works for testing)
  4. Upload a job description file
  5. Watch console for debug messages

Console Commands

Check if user is logged in:

localStorage.getItem('jobsync_current_user')

Check resume analyses:

JSON.parse(localStorage.getItem('jobsync_current_user')).resumeAnalyses

Clear all data and start fresh:

localStorage.clear(); location.reload();

⚠️ Common Issues:

  • Double prompt: Browser may have autofill or extension interfering
  • Nothing happens: Check console for errors, ensure you're logged in
  • Analysis doesn't run: Verify both files are uploaded (check console logs)
